DTS Enabler

Restores DTS and Dolby TrueHD/MLP audio on rooted LG webOS 25 TVs — including DTS in .mkv, .mp4, .ts and .m2ts. Decodes native discrete 5.1 (no stereo downmix). Enable / disable / uninstall from the app, plus a self-test and play-by-ear. No firmware is modified — everything is reversible bind-mounts.

Requires the Homebrew Channel with root. Not endorsed by LG.
