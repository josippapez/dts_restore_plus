Actions

Enable stages the payload and installs the boot hook for the detected profile (applied now, no reboot). Disable removes the boot hook and unmounts (LG's own files are never modified — fully reversible). Uninstall also removes all on-device state.

This TV does not match a supported profile, so Enable is refused. Only webos25-armel-gst124 (verified) and cx-armv7-gst114 (unverified) mechanisms are implemented.

Test the patch

Run self-test decodes a bundled DTS sample in each container (mp4, ts, m2ts) through the TV's media pipeline and reports PASS/FAIL — an objective check that the patch works, no speakers needed. Play plays a sample so you can confirm by ear.
